# Вычислитель отличий

## Установка: ##
```
  git clone https://github.com/dzencot/project-lvl2-s13
  cd project-lvl2-s13
  make install
```

## Использование: ##
```
  make run 1=first_file 2=second_file
```
  or:
```
  make build
  node dist/bin/gendiff.js --help
```
